# 📊 Customer Churn Prediction using Python

## 📌 Project Overview
Customer churn is one of the biggest challenges for businesses, especially in telecom, banking, and subscription-based services. This project focuses on analyzing customer data and building a machine learning model to predict whether a customer is likely to leave the service.

By identifying potential churn customers, companies can take proactive steps to improve customer retention.

---

## 🎯 Objectives

- Analyze customer behavior and service usage
- Identify factors that contribute to customer churn
- Build a predictive model to classify churn vs non-churn customers
- Provide insights to help businesses reduce customer attrition

---

## 🛠️ Tools & Technologies Used

- Python
- Pandas – Data cleaning and manipulation
- NumPy – Numerical operations
- Matplotlib & Seaborn – Data visualization
- Scikit-learn – Machine learning models
- Jupyter Notebook – Development environment

---

## 📂 Dataset Description

The dataset includes customer-related information such as:

- Customer ID
- Gender
- Age
- Tenure
- Contract type
- Monthly charges
- Total charges
- Internet service type
- Payment method
- Churn status (Yes / No)

---

## 🔎 Project Workflow

### 1️⃣ Data Cleaning & Preprocessing
- Handled missing values
- Converted categorical variables to numerical format
- Removed unnecessary columns
- Standardized dataset for analysis

### 2️⃣ Exploratory Data Analysis (EDA)
- Analyzed churn distribution
- Examined relationship between tenure and churn
- Studied impact of monthly charges on churn
- Identified patterns in contract type and customer retention

### 3️⃣ Feature Engineering
- Encoded categorical variables
- Scaled numerical features
- Selected relevant variables for model training

### 4️⃣ Model Building
Machine learning models used:

- Logistic Regression
- Decision Tree
- Random Forest

### 5️⃣ Model Evaluation
Performance evaluated using:

- Accuracy
- Confusion Matrix
- Precision and Recall
- F1 Score

---

## 📊 Key Insights

- Customers with **shorter tenure** are more likely to churn
- **Month-to-month contracts** have higher churn rates
- Higher **monthly charges** increase churn probability
- Customers with **long-term contracts** are more loyal
